# 🤖 DonMatteo AI-Tiket — Модуль интеллектуальной поддержки

## Описание

**DonMatteo AI-Tiket** — это модуль интеллектуальной технической поддержки для Telegram-ботов. Он автоматически обрабатывает обращения пользователей с помощью AI, создаёт топики в Forum-группе для менеджеров и обеспечивает бесшовную коммуникацию между клиентами и командой поддержки.

## Возможности

- 🤖 **AI-ассистент** — автоматически отвечает на вопросы пользователей используя базу знаний (FAQ)
- 📚 **База знаний (FAQ)** — статьи с ключевыми словами для контекстного ответа AI
- 🖼️ **Медиа-вложения** — AI может отправлять фото/видео из FAQ в ответах
- 🎯 **Forum-топики** — каждый тикет создаёт отдельный топик в Telegram Forum-группе
- 👨‍💻 **Менеджеры** — могут отвечать через топик, ответы автоматически доставляются пользователю
- 🔄 **Мульти-провайдер** — поддержка Google, OpenAI, Anthropic, Groq, OpenRouter
- 📊 **Ротация ключей** — автоматический failover между провайдерами и ключами

## Установка и настройка

### 1. Выбор режима AI-Tiket

1. Откройте **Админ-панель → Поддержка → Настройки поддержки**
2. Нажмите **🤖 DonMatteo AI-Тикет**
3. Режим активирован — появятся кнопки настроек AI

### 2. Настройка Forum Group ID

**Что это?** ID Telegram-группы с включенными темами (Topics), куда будут пересылаться тикеты.

**Как получить:**
1. Создайте группу в Telegram
2. Перейдите в **Настройки группы → Темы → Включить**
3. Добавьте бота в группу с правами администратора
4. Добавьте бота `@getmyid_bot` в группу
5. Скопируйте **Chat ID** (формат: `-100xxxxxxxxxx`)

**Настройка в боте:**
1. **Настройки поддержки → 🎯 Forum Group ID**
2. Нажмите **✏️ Изменить Forum ID**
3. Введите ID группы
4. Нажмите **🧪 Проверить подключение** — должен создаться тестовый топик

### 3. Настройка AI-провайдера

1. **Настройки поддержки → 🤖 AI-Провайдеры**
2. Выберите провайдера (Google, OpenAI, Groq и др.)
3. Нажмите **➕ Добавить ключ**
4. Введите API-ключ
5. Нажмите **✅ Включить**
6. Выберите модель (опционально)

**Поддерживаемые провайдеры:**
| Провайдер | Модели | Особенности |
|-----------|--------|-------------|
| Google | gemini-2.5-flash, gemini-pro | Бесплатный тир |
| OpenAI | gpt-4o, gpt-4o-mini | Высокое качество |
| Anthropic | claude-3-sonnet | Безопасные ответы |
| Groq | llama-3.3-70b, mixtral-8x7b | Быстрые ответы |
| OpenRouter | Любые | Универсальный доступ |

### 4. Настройка системного промпта

1. **Настройки поддержки → 💬 Системный промпт**
2. Нажмите **✏️ Редактировать промпт**
3. Введите инструкции для AI
4. Промпт автоматически дополняется FAQ-статьями

**Пример промпта:**
```
Ты AI-ассистент технической поддержки VPN-сервиса "МойVPN".

ПРАВИЛА:
- Отвечай вежливо и кратко
- Используй базу знаний для точных ответов
- Если не знаешь ответ — предложи позвать менеджера
- Никогда не выдумывай информацию

КОНТЕКСТ:
- Сервис: VPN для всех устройств
- Поддержка: WireGuard, VLESS, Shadowsocks
```

### 5. Создание FAQ-статей

1. **Настройки поддержки → 📚 FAQ-Статьи**
2. Нажмите **➕ Добавить статью**
3. Введите:
   - **Заголовок** — краткое название
   - **Содержание** — полный ответ
   - **Ключевые слова** — для поиска (через запятую)
4. Опционально: добавьте медиа (фото/видео)

**Медиа-вложения:**
- Добавьте медиа через **📎 Добавить медиа**
- Укажите уникальный тег: `setup_android`, `error_connection`
- В ответе AI напишите `[media:setup_android]` — медиа отправится автоматически

## Как это работает

### Флоу пользователя

```
1. Пользователь нажимает "Создать обращение"
   ↓
2. Вводит сообщение
   ↓
3. Создаётся Forum-топик (если настроен forum_id)
   ↓
4. AI анализирует вопрос + FAQ контекст
   ↓
5. AI отвечает пользователю
   ↓
6. Сообщение копируется в Forum-топик
   ↓
7. Пользователь может:
   - Продолжить диалог с AI
   - Нажать "🆘 Вызвать менеджера"
   - Посмотреть историю в "🎫 Мои обращения"
```

### Флоу менеджера

```
1. Менеджер видит новый топик в Forum-группе
   ↓
2. Отвечает в топике
   ↓
3. Бот пересылает ответ пользователю
   ↓
4. AI автоматически отключается
   ↓
5. Менеджер может:
   - Включить AI обратно: кнопка "🤖 Включить AI"
   - Закрыть тикет: кнопка "✅ Закрыть тикет"
   - Использовать команды: /close, /ai_on, /ai_off
```

|---------|----------|

## API конфигурации

### Переменные окружения (опционально)

```env
# ID Forum-группы (можно настроить через бота)
SUPPORT_AI_FORUM_ID=-100xxxxxxxxxx

# Включить AI (по умолчанию True при выборе режима ai_tiket)
SUPPORT_AI_ENABLED=True

# Режим поддержки: tickets, contact, both, ai_tiket
SUPPORT_SYSTEM_MODE=ai_tiket
```

### Таблицы базы данных

| Таблица | Описание |
|---------|----------|
| `forum_tickets` | Тикеты (user_id, topic_id, status, ai_enabled) |
| `forum_ticket_messages` | История сообщений с медиа |
| `ai_faq_articles` | FAQ-статьи |
| `ai_faq_media` | Медиа-вложения FAQ |
| `ai_provider_configs` | Настройки AI-провайдеров |

### Ручная интеграция

1. Скопируйте `app/modules/ai_ticket/` в проект
2. Скопируйте `app/database/models_ai_ticket.py`
3. Добавьте миграцию `ai_ticket_001_initial.py`
4. Добавьте в `bot.py`:
   ```python
   from app.handlers.admin import support_settings as admin_support_settings
   from app.modules.ai_ticket.handlers.manager import register_manager_handlers
   
   # В функции регистрации хендлеров:
   admin_support_settings.register_handlers(dp)
   register_manager_handlers(dp)
   ```
5. Добавьте в `tickets.py` маркеры интеграции
6. Добавьте локализацию в `locales/*.json`

## Решение проблем

### AI не отвечает
- Проверьте, добавлен ли API-ключ провайдера
- Проверьте, включен ли провайдер
- Проверьте логи: `ai_manager.all_keys_exhausted`

### Ответы менеджера не доходят
- Проверьте, настроен ли Forum Group ID
- Проверьте, что бот — администратор группы
- Проверьте логи: `ai_ticket_manager`

### Пользователь не видит тикеты
- Убедитесь, что режим `ai_tiket` активен
- Проверьте, что тикет создавался в режиме AI

### Топики не создаются
- Проверьте права бота в группе
- Включены ли темы (Topics) в группе
